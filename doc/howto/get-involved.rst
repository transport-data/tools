Get involved with the TDC(I)
****************************

There are many ways to get involved with the Transport Data Commons (TDC)
and TDC Initiative (TDCI).
This guide gives a long list of just some of these ways,
arranged from ‘A’ to ‘Z’.
Expand the right navigation sidebar here to see a list of all the headings.

.. _gi-browse:

Browse data
===========

- *Visit* `portal.transport-data.org/datasets <https://portal.transport-data.org/datasets>`_.
- *Use* the search box, featured links, and other features to identify data sets.
- *Explore* the metadata and attached files with data and documentation.
- *Follow* the metadata > URL links to upstream sources.

.. _gi-zulip:

Chat on Zulip
==============

`transport-data.zulipchat.com <https://transport-data.zulipchat.com>`_
is the TDC's sponsored instance of the Zulip open-source chat platform.

- *Register* an account for free.
- *Introduce* yourself to other members of the community;
  *read* others' introductions.
- *Read* or *join* existing discussions.

.. _gi-create:

Create records via the Portal
=============================

- *Read* HOWTO :doc:`/howto/portal`.
- *Try* it yourself!

  - Create a record with *metadata only*,
    describing an upstream data source you are aware of or rely on.
  - Create a record and *upload data files*
    for data that you or your organization provide or maintain.

.. _gi-portal-org:

Create your organisation
========================

‘Organisations’ on the TDC Portal collect users and datasets,
and align with real-world organisations that act as data providers
or publishers.

- :ref:`gi-register`.
- *Create* an organisation, filling in its name and other information.
- *Get approval* from TDCI admins.
- *Invite* your colleagues to join this organisation as well.

.. _gi-edit-docs:

Edit these HOWTOs
=================

All TDC code and documentation is open source.
The HOWTO page you are reading is built from a file on GitHub;
see `github.com/transport-data/tools <https://github.com/transport-data/tools/tree/main/doc/howto>`_.

- *Register* an account at `github.com <https://github.com>`_.
- *Use* the GitHub website to edit files directly,
  or use Git to clone, edit, commit, and push your changes.
- *Open* a ‘pull request’ to merge your updates and improvements into the current version.

.. _gi-metadata:

Improve metadata
================

Complete, up-to-date, and detailed metadata—information *about* data—is very important in the TDC.
It allows people to quickly see whether and how they could use data in their work.

- *Read* and *learn* HOWTO :doc:`metadata`.

.. _gi-meetings:

Join a meeting
==============

The TDCI organizes two kinds of regular meetings:

- “Data Group” meetings are held biweekly,
  last about 60 minutes,
  and focus on particular data work and tools improvements.
- “TDC Global” meetings are held every 2–3 months,
  last about 90 minutes,
  and focus on general topics, including non-technical topics.

To find out about these meetings,
please watch for announcements in the #tdci-general Zulip channel.

Register for the Portal
=======================

- *Register* an account at github.com_.

  button on transport-data.org_ to log in with your GitHub account.

.. _gi-share:

Share the TDC with colleagues
=============================

- *Identify* 1 or a few colleagues with whom you share an interest in transport data.
  For example, think of others who are interested in transport and mobility where you are,
  and how to measures its features, problems, and possibilities.
- *Reach* out to them to share (a) links to `transport-data.org <https://transport-data.org>`_,
  this page, and your favourite TDC resources, plus
  (b) what excites and interests you about TDC activities.

.. _gi-glasgow-decl:

Sign the Glasgow Declaration
============================

- *Read* the `Glasgow Declaration on a Transport Data Commons <https://docs.google.com/document/d/1sFLG9mBvWYL4UZQ4eUQ3fqKqcXGJDut-ZxwwxJQV_w0>`_.
- *Sign* on behalf of your organization or yourself.
- *Share* that you've signed on social media.

.. _gi-showroom:

View data visualizations
========================

- Visit the ‘Showroom’ page on the portal:
  `transport-data.org/showroom <https://portal.transport-data.org/showroom>`_.
- *See* how TDC data can be visualized and used in other useful data products.

.. _gi-use-data:

Use data via the TDC
====================

- First :ref:`gi-browse`.
- *Download* and open one or more data files.
- *Use* your favourite tools and programs to explore the data,
  produce a plot or compare some values.
- *Share* your results, and cite the TDC and (if any) upstream data source.

.. _gi-code:

Use and extend these tools
==========================

The TDCI maintains the :mod:`transport_data` Python package,
which includes a collection of tools for handling data
and interacting with the Portal.

- :ref:`Install <install>` the package.
- *Browse* the documentation.
- *Use* the code to, for example:

  - :doc:`cli-edit`.
  - *Write* code of your own to transform (meta)data you produce or use.
